# BOM — Micro Travel Kit (Wire C)

## Bill of Materials

| Item | Spec | Qty | Source | ~Cost |
|---|---|---|---|---|
| Silicone-insulated stranded wire | 28 AWG, hi-vis color (matching A and B) | ~13 ft | BNTechgo on Amazon | $2 (from existing spool, or $5 for new spool) |
| 3 mm spade lug (M3 / #4) | crimp, 22-18 AWG barrel | 2 (1 + spare) | local | $1 |
| #4 ring terminal (alternative to soldered eyelet) | crimp | 2 (optional) | local | $0.50 |
| Heat-shrink tubing | 3 mm and 6 mm | as needed | from existing kit | $0 |
| Small winder | 60 × 30 mm, accepts 13' of wire | 1 | print yourself or use a fishing-line spool | $0 (3D print) or $2 |
| **Subtotal** | | | | **~$3** |

If you bought enough materials for the counterpoise wire kit, this kit is essentially free — just an additional 13' of wire and one spade lug from the existing supplies.

## Field Spares (Recommended in the Pouch)

| Item | Qty | Why |
|---|---|---|
| In-line banana plug + socket pair | 1 set | the chain joint between A and B is the likely failure point in Option 3; field replacement saves the activation |
| Heat-shrink kit (small) | 1 | repair frayed spots or reseat a worn termination |
| 3 mm spade lugs | 2 | replace a damaged termination |
| Tent stake | 1 | for staking down Wire C's eyelet end |
| Hi-vis flagging tape | small piece | tie to Wire C's eyelet for visibility |

## Total Kit Weight (built)

| Item | Weight |
|---|---|
| Wire C (13 ft + lug + eyelet) | ~3 g |
| Small winder | ~5 g |
| Field spares (banana pair + heat-shrink + 2 spades + tent stake + flag) | ~10 g |
| **Total** | **~18 g** |

This is the "Option 3 add-on" bring-along. Combined with the
counterpoise-wire-kit (~12 g) and the BNC-to-binding-post adapter
(~5 g), the total wire-kit footprint for all three direct-to-BNC
options is ~35 g.

## Sourcing Notes

If you're 3D printing winders for both this kit and the
counterpoise-wire-kit, print them at the same time. Adam Kimmerly's
K6ARK winder STL on Printables (linked in the parent project notes)
has variants for different wire lengths.
